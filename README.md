# 用前准备

node 版本 14.18+ 或 16+ 或更高

`npm (yarn、pnpm) i (add) byl-cli -g`

`npx byl-cli create <project-name>`

# 查看命令详情

`byl` | `byl --help` | `byl -h`

# 查看版本

`byl --version` | `byl -V`

# 新建项目

`byl create <project-name> -f (--force)`

例如：`byl create vue-template -f`

# 查看命令详情

`byl <command> --help`

例如：`byl create --help`
。
